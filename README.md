# cordis
The next generation of Cordis.

## Run & Build
```bash
python3 -m venv venv
source venv/bin/activate
pip3 install -r requirements.txt
cd frontend
npm run build
cd ..
sudo bash init-postgresql.sh
sudo bash start.sh
```
